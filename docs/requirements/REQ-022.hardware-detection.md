# REQ-022: Hardware Auto-Detection and Resource Profiles

## Description

`deploy.sh` automatically detects hardware capabilities and configures resource limits accordingly. This ensures the system runs well on both constrained (Pi 3) and powerful (Pi 5, x86_64) hardware.

## Detection

| Property | Method | Impact |
|----------|--------|--------|
| Architecture | `uname -m` | Skip Tidal on x86_64 |
| Total RAM | `/proc/meminfo` | Select resource profile |
| NFS mounts | `mount \| grep nfs` | Increase MPD start period |
| Framebuffer | `/sys/class/graphics/fb0` | Enable TUI progress display |

## Resource Profiles

Environment variables are set per-service in `.env`:
- `{SERVICE}_MEM_LIMIT`, `{SERVICE}_MEM_RESERVE`, `{SERVICE}_CPU_LIMIT`
- Services: SNAPSERVER, AIRPLAY, SPOTIFY, MPD, MYMPD, METADATA, TIDAL

## Dependencies

- Linux kernel with `/proc/meminfo` and `/proc/cpuinfo`
