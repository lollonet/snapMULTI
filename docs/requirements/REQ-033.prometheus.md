# REQ-033: Prometheus Metrics

## Description

Expose container health, audio stream status, and system metrics via a Prometheus-compatible endpoint for monitoring and alerting.

## Proposed Metrics

| Metric | Type | Description |
|--------|------|-------------|
| snapmulti_container_healthy | Gauge | 1=healthy, 0=unhealthy per container |
| snapmulti_stream_active | Gauge | 1=playing, 0=idle per source |
| snapmulti_clients_connected | Gauge | Number of connected snapclients |
| snapmulti_cpu_usage_percent | Gauge | Per-container CPU usage |
| snapmulti_memory_usage_bytes | Gauge | Per-container memory usage |
