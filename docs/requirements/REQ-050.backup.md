# REQ-050: Backup and Restore

## Description

Provide scripts to backup and restore snapMULTI configuration, persistent state, and metadata. Essential for SD card reliability (corruption risk) and migration between devices.

## Proposed Scope

### Backup Contents
| Item | Path | Priority |
|------|------|----------|
| Environment config | `.env` | Critical |
| Snapserver config | `config/` | Critical |
| MPD database | `mpd/data/mpd.db` | High |
| MPD playlists | `mpd/data/playlists/` | High |
| myMPD state | `mympd/` | Medium |
| Artwork cache | `artwork/` | Low |

### Excluded
- Docker images (re-pulled on restore)
- Music files (external storage)
- Logs (ephemeral)

## Proposed Behavior

1. `scripts/backup.sh` creates timestamped tar.gz
2. `scripts/restore.sh` extracts and restarts services
3. Optional: scheduled backup via cron/systemd timer
4. Optional: remote backup to NFS/SMB share
